# Linux Notes

## Command Line Basics
- `ls` – list files
- `cd` – change directory
- `pwd` – print working directory
- `mkdir` – make a new folder
- `touch` – create a file

## Permissions
- `chmod` – change file permissions
- `chown` – change ownership
- Example: `chmod 755 script.sh`
